HTML El html es el lenguaje estándar para crear y estructurar contenido en la web. Permite organizar texto, imágenes, vídeos y otros elementos en una página web, facilitando la creación de sitios web visuales y funcionales. Tambien usa el lenguaje de etiquetas para definir los diferentes elementos de una página. Cada elemento HTML se cierra en etiquetas.
